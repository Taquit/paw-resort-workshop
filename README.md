# Paw Resort: AWS Containers Workshop

Welcome to the **Paw Resort Workshop**! This repository contains the source code for a lightweight veterinary management system, built specifically to practice container orchestration and cloud deployments.

The goal of this project is to take a simple local application and successfully deploy it to the cloud using serverless container architecture.

## Learning Objectives
By completing this workshop, you will learn how to:
* Containerize a backend (FastAPI) and frontend (HTML/Bootstrap) using **Docker**.
* Push container images to **Amazon ECR** (Elastic Container Registry).
* Create and configure **Task Definitions** for cloud environments.
* Deploy serverless containers using **Amazon ECS with AWS Fargate**.
* Expose the application to the internet securely using an **Application Load Balancer (ALB)**.

## Tech Stack
* **Frontend:** HTML5, Bootstrap 5
* **Backend:** Python, FastAPI
* **Containerization:** Docker, Docker Compose
* **Cloud Infrastructure:** AWS (ECR, ECS, Fargate, ALB, VPC)
